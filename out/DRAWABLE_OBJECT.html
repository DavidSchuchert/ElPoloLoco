Class: DRAWABLE_OBJECT

DRAWABLE_OBJECT()

Represents a drawable object in the game with image-rendering capabilities. This class provides the ability to load, cache, and draw images onto a canvas context. It also has an optional bounding box drawing function which works for certain derived classes.
