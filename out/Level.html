Class: Level

Level(enemies, endboss, bottles, coins, clouds, backgroundObjects)

Represents a game level. This class holds all the elements that construct a particular level, such as enemies, endboss, coins, clouds, and other background objects.

new Level(enemies, endboss, bottles, coins, clouds, backgroundObjects)

Creates a new game level.
