Represents the current state of keyboard inputs. The `Keyboard` class provides a simple way to track the state of specific keys. Each key has a boolean value representing whether it's currently pressed (true) or not pressed (false).
Properties:
Name Type Description
LEFT boolean Represents the state of the LEFT arrow key.
RIGHT boolean Represents the state of the RIGHT arrow key.
UP boolean Represents the state of the UP arrow key.
DOWN boolean Represents the state of the DOWN arrow key.
SPACE boolean Represents the state of the SPACE key.
D boolean Represents the state of the D key.
