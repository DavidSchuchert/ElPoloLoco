(constant) level1 :Level

Represents the first level of the game. This instantiation sets up various objects within the level such as enemies, end boss, bottles, coins, clouds, and background objects. Each object type is passed as an array to create multiple instances of the same type.
Type:
  • Level
Properties:
Name Type Description
chickens Array.<Chicken> Array of Chicken enemies for the level.
endboss Array.<Endboss> Array containing the end boss for the level.
bottles Array.<Bottles> Array of Bottles collectibles for the level.
coins Array.<Coins> Array of Coins collectibles for the level.
clouds Array.<Cloud> Array containing a Cloud object for the level.
backgrounds Array.<BackgroundObject> Array of background objects to set the visual theme of the level.
